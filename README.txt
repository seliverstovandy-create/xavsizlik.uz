# TechSupply Uzbekistan — static site

Готовый статический сайт для публикации на:
- Cloudflare Pages
- Netlify
- GitHub Pages
- Vercel
- любом обычном static hosting

## Файлы
- `index.html`
- `styles.css`

## Как опубликовать
### Вариант 1: Cloudflare Pages / Netlify
Загрузите содержимое этой папки как статический сайт.

### Вариант 2: GitHub Pages
Загрузите файлы в репозиторий и включите GitHub Pages.

## Что заменить
В `index.html` замените:
- название компании
- адрес
- телефоны
- e-mail
- ИНН / реквизиты
- текст о компании

## Примечание
Форма сейчас демонстрационная. Для реальной отправки заявок можно подключить:
- Formspree
- Netlify Forms
- Telegram bot
- email backend
- CRM / API
